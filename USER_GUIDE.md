# Steam 游戏智能推荐助手 - 使用文档

## 📖 简介

Steam 游戏智能推荐助手是一个基于 AI 的 MCP (Model Context Protocol) 服务，为 Claude Desktop 等支持 MCP 的应用提供强大的 Steam 游戏推荐和查询功能。

**服务地址：** `https://steam-mcp.onrender.com/mcp`

## ✨ 主要功能

### 🎯 1. 智能游戏推荐 (recommend_games)
根据您的需求描述，使用 AI 智能匹配和推荐最适合的游戏。

**适用场景：**
- "我想玩开放世界 RPG 游戏，预算 100 元以内"
- "推荐几款支持多人联机的射击游戏"
- "有什么适合休闲放松的独立游戏？"

**参数：**
- `user_query` (必填): 您的游戏需求描述
- `max_results` (可选): 返回游戏数量，默认 5 款（建议 ≤10）

**特点：**
- ✅ AI 智能分析需求
- ✅ 精准匹配游戏
- ✅ 提供详细推荐理由
- ⚠️ 响应较慢（1-3分钟），但结果最精准

**示例：**
```
推荐 5 款适合新手的开放世界 RPG 游戏，预算 150 元
```

---

### 🔍 2. 快速游戏搜索 (search_games)
快速搜索 Steam 游戏，无需 AI 分析，响应速度快。

**适用场景：**
- 已知游戏类型或关键词
- 需要快速查询结果
- 预算有限的游戏搜索

**参数：**
- `keywords` (必填): 搜索关键词，如 "open world rpg"、"射击游戏"
- `max_price` (可选): 最高价格（人民币），如 100.0
- `max_results` (可选): 返回数量，默认 10 款

**特点：**
- ⚡ 响应速度极快（秒级）
- ✅ 支持价格筛选
- ✅ 适合快速浏览

**示例：**
```
搜索关键词为 "horror survival" 的游戏，价格不超过 80 元
```

---

### 🎁 3. 折扣游戏查询 (get_discounted_games)
获取当前正在打折的 Steam 游戏，把握优惠时机。

**适用场景：**
- Steam 促销季节
- 寻找性价比高的游戏
- 预算有限但想多买几款

**参数：**
- `min_discount` (可选): 最低折扣百分比（0-100），默认 0
  - 例如：50 表示至少 5 折（50% off）
- `max_price` (可选): 折后最高价格（人民币）
- `max_results` (可选): 返回数量，默认 20 款

**特点：**
- 🏷️ 实时获取折扣信息
- 📊 按折扣力度排序
- 💰 帮您省钱

**示例：**
```
查找折扣至少 70% 的游戏，价格不超过 50 元
```

---

### 📖 4. 游戏详情查询 (get_game_details)
获取单个游戏的完整详细信息。

**适用场景：**
- 了解游戏具体内容
- 查看系统要求
- 对比不同游戏

**参数：**
- `game_identifier` (必填): 游戏名称或 Steam AppID
  - 例如："艾尔登法环" 或 "1245620"

**返回信息包括：**
- 📝 游戏名称、描述
- 💵 价格、折扣信息
- 🏢 开发商、发行商
- 🎮 游戏类型、标签
- 🌍 支持语言
- 💻 系统要求
- ⭐ 评分信息
- 📸 游戏截图链接

**示例：**
```
获取"黑神话：悟空"的详细信息
```

---

### 🔥 5. 热门游戏排行 (get_top_games)
查看 Steam 当前最热门的游戏排行榜。

**适用场景：**
- 发现当前流行游戏
- 了解市场趋势
- 寻找热门大作

**参数：**
- `max_results` (可选): 返回数量，默认 20 款
- `filter_type` (可选): 排行榜类型，默认 'topsellers'
  - `topsellers`: 畅销榜（最受欢迎）
  - `popularnew`: 热门新品
  - `trendingweek`: 本周热门趋势

**示例：**
```
查看 Steam 本周热门游戏前 10 名
```

---

### 🆓 6. 免费游戏列表 (get_free_games)
发现优质的免费游戏。

**适用场景：**
- 零预算玩家
- 试玩体验
- 寻找免费精品游戏

**参数：**
- `max_results` (可选): 返回数量，默认 20 款
- `tags` (可选): 游戏标签过滤，如 ["动作", "冒险", "多人"]

**示例：**
```
查找免费的多人在线游戏
```

## 🚀 快速开始

### 方式 1: 在 Claude Desktop 中使用（推荐）

1. **找到配置文件**
   - Windows: `%APPDATA%\Claude\claude_desktop_config.json`
   - macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`

2. **编辑配置文件**
   ```json
   {
     "mcpServers": {
       "steam-recommender": {
         "transport": "sse",
         "url": "https://steam-mcp.onrender.com/mcp"
       }
     }
   }
   ```

3. **重启 Claude Desktop**

4. **开始使用**
   直接在对话中说：
   - "帮我推荐几款开放世界游戏"
   - "Steam 现在有什么好的折扣游戏？"
   - "查看《艾尔登法环》的详细信息"

## 💡 使用技巧

### 智能推荐最佳实践

1. **明确需求**
   ✅ "推荐适合新手的第三人称射击游戏，预算 100 元"
   ❌ "推荐游戏"

2. **提供约束条件**
   - 预算范围
   - 游戏类型
   - 特殊需求（多人、单机、剧情向等）

3. **控制数量**
   - 推荐功能建议 ≤10 款（避免等待过久）
   - 快速搜索可以多一些

**组合搜索：**
- "open world RPG"

## ⚠️ 注意事项

### 服务限制

1. **免费版休眠机制**
   - 服务闲置 15 分钟后会休眠
   - 休眠后首次访问需要等待 30-60 秒唤醒
   - **解决方案**：耐心等待或使用 UptimeRobot 等工具保持活跃

2. **响应时间**
   - 智能推荐：1-3 分钟（使用 AI 分析）
   - 快速搜索：秒级响应
   - 其他功能：通常 < 10 秒

3. **价格信息**
   - 价格仅供参考，以 Steam 实际价格为准
   - 折扣信息实时获取，可能存在延迟

### 使用建议

- ✅ 首次使用建议用"快速搜索"测试服务是否正常
- ✅ 智能推荐适合需要详细分析的场景
- ✅ 折扣查询适合促销季节使用
- ⚠️ 避免在高峰期同时发起多个智能推荐请求

## 🔧 技术信息

**技术栈：**
- 框架: FastMCP 2.13.1
- AI 模型: 阿里云通义千问 (Qwen-Plus)
- 数据来源: Steam 官方 API
- 部署平台: Render (免费版)

**传输协议：** SSE (Server-Sent Events)

**开源地址：** https://github.com/Mouseminar/steam_mcp

**问题反馈：**

**Q: 智能推荐为什么需要这么久？**
A: 每个游戏都需要 AI 分析匹配度，建议控制在 10 款以内。

**Q: 价格信息准确吗？**
A: 实时抓取，但可能有延迟，请以 Steam 官方为准。

**Q: 支持哪些地区？**
A: 主要支持中国区 Steam，价格为人民币。
